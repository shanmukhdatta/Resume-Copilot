      Questions

      Good to know

      Can it invent experience I don't have?

      No. Every prompt instructs agents to use only your parsed resume data, and a separate validation pass fact-checks the output before export — unsupported claims trigger a retry of just that section.

      What files can I upload?

      Resumes as PDF or DOCX. Job descriptions as PDF, DOCX, or plain text.

      Where does this run?

      This page talks to your own backend instance over the API — configure the URL in js/config.js. Nothing is sent anywhere else.

      What export formats are supported?

      PDF, DOCX, and HTML, in any of four templates.
